• Loop Algorithm

    サンプルとして与えられた。

  • Catmull Clark Algorithm

    次の点を計算するには、頂点周りの多角形の重心や隣の頂点との中点しか必要ないので、
    入力が一般のポリゴンであっても適用することが可能であるアルゴリズムである。
    新たな頂点のidは元の頂点のidに対応させ、
    辺の中点は(元の頂点数 + edge.id)、多角形の重心のidを(元の頂点数 + 元の辺数 + face.id)とした。

  • Doo Sabin Algorithm

    頂点と辺の中点2つと重心の合計4点によりできる四角形を見出すことで
    一般のポリゴンに対しても同じように計算することができるアルゴリズムになっている。
    ハーフエッジ構造は、辺に矢印を書いて考えると実装しやすい。
    新たにできる頂点のidは、ハーフエッジのidそのままで対応させられることに気がつくのに手間取った。
